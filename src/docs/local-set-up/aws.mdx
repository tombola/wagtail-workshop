# AWS Account and CLI

Go to the [Amazon Web Services Website](https://aws.amazon.com/?nc2=h_lg)

![aws website](./aws-creat.png)

Click on 'Create Account' and create an account.

Install the [AWS CLI](https://docs.aws.amazon.com/cli/latest/userguide/cli-chap-install.html). This will allow you to interact with AWS from your commandline.
